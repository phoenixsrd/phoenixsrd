## 🚀 Meus Projetos

Aqui estão **todos** os projetos públicos que já desenvolvi no GitHub:

### [pulsonews](https://github.com/phoenixsrd/pulsonews)
**Agregador de notícias em tempo real — PULSO NEWS**
> Notícias ao vivo com atualização automática a cada 5 minutos.

- Botões **AO VIVO**, **ATUALIZAR** e **▶ URGENTE**
- Auto-refresh + contagem regressiva
- Interface limpa e focada em conteúdo fresco

**Demo ao vivo**: [pulsonews.vercel.app](https://pulsonews.vercel.app)  
**Linguagem principal**: HTML + JavaScript

### [yt-dlp-advanced](https://github.com/phoenixsrd/yt-dlp-advanced)
**Downloader avançado de vídeos e áudios**
> Projeto completo para baixar conteúdo de +1000 sites usando **yt-dlp**.

- Interface CLI interativa + servidor web
- Suporte a playlists (com skip/intervalo), múltiplos formatos (mp4, mp3, mkv, webm, opus)
- Sistema de fila, downloads simultâneos, proxy/VPN e atualização automática
- Extração de legendas, thumbnails e metadados JSON

**Linguagem principal**: Python

### [twitchmonitor](https://github.com/phoenixsrd/twitchmonitor)
**Monitor de canais da Twitch**
> Script que verifica em intervalos regulares se um canal está ao vivo.

- Usa a API oficial da Twitch (Client ID + Secret)
- Notificações e execução automática no boot (Windows, Linux, macOS)
- Configuração simples e leve

**Linguagem principal**: Python

### [autodowm](https://github.com/phoenixsrd/autodowm)
**Visualizador e downloader de releases do GitHub**
> Página web simples que puxa automaticamente as releases de qualquer repo público e oferece links diretos para download.

- Totalmente customizável (basta trocar owner e repo no código)
- Interface limpa e funcional

**Demo ao vivo**: [autodowm.vercel.app](https://autodowm.vercel.app)  
**Linguagem principal**: HTML + JavaScript
